# ycraft
my top down sandbox game

## compile
```
$ make
```

a binary will be placed in the `bin` folder

### dependencies
SDL2, SDL2_ttf, SDL2_image, SDL2_mixer, libcurl

## controls
WASD - move player

left click - delete block

right click - place block

t - open chat

e - open/close inventory
